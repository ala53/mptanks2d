Seedlings help

On this page you'll find help for the game Seedlings

What is Seedlings?

Seedlings is an example game made by Player.IO to showcase their Sitebox service for easily building Facebook Games.

Can i play it?

Sure, but the game has limited content and it's main function is not to provide hours and hours of fun gameplay, but rather to showcase that fully-featured Facebook Games can easily be built with Sitebox.
